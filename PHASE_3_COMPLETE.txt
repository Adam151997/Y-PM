## 💡 Usage Examples

### Using Breadcrumbs
```tsx
// Automatic from pathname
<Breadcrumbs />

// Manual configuration
<Breadcrumbs items={[
  { label: 'Projects', href: '/projects', icon: <Folder /> },
  { label: 'Website Redesign', href: '/projects/1', icon: <Folder /> },
]} />
```

### Using Page Transitions
```tsx
<PageTransition>
  <DashboardContent />
</PageTransition>

// Or simpler
<FadeTransition>
  <ModalContent />
</FadeTransition>
```

### Using Skeleton Loading
```tsx
// While loading
{isLoading ? (
  <SkeletonList count={5} />
) : (
  <TaskList tasks={tasks} />
)}

// Dashboard loading
<SkeletonDashboard />
```
