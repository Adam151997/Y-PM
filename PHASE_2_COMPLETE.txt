GLOBAL
  ⌘K / Ctrl+K ........... Open command palette
  ⌘N / Ctrl+N ........... Create new project
  ⌘\ / Ctrl+\ ........... Toggle theme
  ? ....................... Show all shortcuts
  Esc ..................... Close dialogs

NAVIGATION (Gmail/Linear style)
  G then D ............... Go to Dashboard
  G then P ............... Go to Projects
  G then A ............... Go to Activities
  C ....................... Create new (quick)

COMMAND PALETTE
  ↑↓ ...................... Navigate items
  ↵ ....................... Select item
  Esc ..................... Close palette
  Type .................... Search/filter
